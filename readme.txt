AAllows you to approve your WordPress site comments from your inbox with one-click

== Description ==

This plugin incorporates the Gmail action API to WordPress comment approval. You can directly
the comments from your Gmail inbox without logging into your site.

== Installation ==

1. Upload `gmail-inbox-action` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
